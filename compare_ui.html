Local Compare Workspace

Drop a DOCX and a saved HTML page. Get back an annotated PDF with embedded difference comments.

This interface runs the local comparison pipeline, renders the webpage through Chromium, and annotates only the meaningful content differences against the Word source.

HTML Proofread Flow Use DOCX versus a saved HTML page for webpage proofs and proofread-style embedded comments.
Drag-and-drop flow Drop files anywhere on their panels or click to pick them manually.
Review-ready output The generated annotated PDF can be downloaded immediately and previewed in the browser.

Inputs

Use one Word `.docx` file and one saved webpage `.html` or `.htm` file. The compare job runs locally on this machine.
